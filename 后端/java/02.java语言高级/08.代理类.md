





> 代理由三部分组成 代理接口 代理类 被代理类



#### 静态代理的实现

```java
//代理接口
interface Factory {
   void production();
}

//代理类
class ProxyFactory implements Factory {
   private Factory factory;
   ProxyFactory(Factory factory) {
      this.factory = factory;
   }
   @Override
   public void production() {
      System.out.println("代理类准备生产");
      factory.production();
      System.out.println("代理做收尾工作");
   }
}

//被代理类
class NikeProxyFactory implements Factory {
   @Override
   public void production() {
      System.out.println("生产了一批衣服");
   }
}


public class ProxyTest {
   @Test
   public  void  test01(){
      //提供被代理类
      NikeProxyFactory nike = new NikeProxyFactory();
      //提供代理类
      ProxyFactory proxyFactory = new ProxyFactory(nike);
      proxyFactory.production();
   }
}
```
