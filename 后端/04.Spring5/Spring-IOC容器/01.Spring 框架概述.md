





### 1.Spring基本概念

1. Spring 是轻量级的开源的 JavaEE 框架 
2. Spring 可以解决企业应用开发的复杂性 
3. Spring 有两个核心部分：IOC容器 和 Aop
   1. IOC容器：
      - IOC容器，又叫Spring容器，Spirng通过这个容器来管理对象
      - 控制反转，把创建对象过程交给 Spring 进行管理，Spring管理着对象的整个生命周期
   2. Aop：面向切面，不修改源代码进行功能增强

### 2.Spring特点

1. 方便解耦，简化开发
2. op 编程支持
3. 方便程序测试
4. 方便和其他框架进行整合
5. 方便进行事务操作
6. 降低 API 开发难度
